HTML static Index

The list of static tests (those that involve in scripting and no user interaction).

Please select the type of test harness you wish to use to embed the tests inside the navigation aids:

Self Contained
Tests consist of an HTML page describing the test and containing, inline, the test content.
XHTML <iframe>
Tests consist of an XHTML wrapper page summarising the test and linking to the actual test content using an <iframe> element.
XHTML <object>
Tests consist of an XHTML wrapper page summarising the test and linking to the actual test content using an <object> element.
XHTML <frame>
Tests consist of a two frame XHTML frameset, the top frame being an XHTML wrapper page summarising the test and the bottom frame being the actual test content.
HTML <iframe>
Tests consist of an HTML wrapper page summarising the test and linking to the actual test content using an <iframe> element.
HTML <object>
Tests consist of an HTML wrapper page summarising the test and linking to the actual test content using an <object> element.
HTML <frame>
Tests consist of a two frame HTML frameset, the top frame being an HTML wrapper page summarising the test and the bottom frame being the actual test content.
XLink embed
Tests consist of an XML page summarising the test and linking to the actual test content using an XLink with the show axis set to embed.
TNG Format
Tests consist of an HTML page with a brief test summary and navigation aids and a link to the test content using an <object> tag. This test format is designed to be stylistically compatible with the TNG test format used for other CSS test suites.
